# ⚽ ScoreSight – AI Powered Football Prediction System

## 📌 Overview

ScoreSight is an AI-powered football analytics web application developed using Machine Learning and Flask. The system predicts:

- Match Scores
- Match Winners
- Player Goals
- Player Assists

using historical English Premier League statistics.

---

## 🚀 Features

- Home Page
- Match Prediction
- Player Prediction
- AI Prediction Summary
- Modern Responsive UI
- Random Forest Regression Model

## 📊 Machine Learning

### Match Prediction

Algorithm:

Random Forest Regressor

Inputs

- Home Team
- Away Team
- Rolling Team Statistics

Outputs

- Home Goals
- Away Goals
- Winner

---

### Player Prediction

Algorithm

Random Forest Regressor

Inputs

- Club
- Position
- Age
- Appearances
- Shots
- Passes
- Tackles
- Recoveries

Outputs

- Predicted Goals
- Predicted Assists
